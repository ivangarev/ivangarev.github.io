Программные средства оперативно - технического анализа сигналов. 

 

1. CoolEdit2000



Программа предназначена для проведения записи, спектрально-временного анализа сигналов, их графического отображения, редактирования, фильтрации. Рекомендации по установке программы: имя - MFD Corp., серийный номер - Q17EXF5U. 

» Установить CoolEdit2000             » Зарегистрировать CoolEdit2000

2. Spectra Plus v.4.00.05



Программа предназначена для записи сигналов, конвертирования звуковых файлов в различные форматы, спектрального, временного анализа сигналов. Множество настроек отображения, имеется встроенный тестовый генератор. При записи сигналов с эфира гораздо удобней CoolEdit. После установки программы на жесткий диск перед первым запуском программы НЕОБХОДИМО записать файл SP405CRK.EXE в подкаталог BIN, где содержится запускной файл SPECPLUS.EXE с характерной иконкой. Далее запускайте файл SP405CRK.EXE. Теперь все - с программы снята защита, можно пользоваться. Устанавливать программу необходимо в корневой каталог диска. 

» Установить SpectraPlus  

3. Афина



Программа предназначена для первичного спектрально-временного анализа, демодуляции и декодирования фазомодулированных передач. Имеется возможность обработки факсов, сеансов ПД. При установке регистрации не требует. 

» Установить СПО Афина

4. Антракт 98



Более ранняя версия СПО "Афина". Однако обладает рядом возможностей, которых нет в "Афине".  Это в частности автоматический подбор режима демодуляции в соответствии с выставленными параметрами. Более удачно решен вопрос отображения временной диаграммы и спектра сигнала. Установка программы требует произведения определенных действий, связанных с подбором кода и взлома серийного номера программы. Порядок взлома описан в файле. Установку СПО "Антракт-98" производить в корневой каталог диска. 

» Установить программу     » Прочитать файл об установке СПО

5. WinTone



Программный продукт WinTone предназначен для декодирования тональных сигналов набора номера. Регистрация программы осуществляется следующим образом: необходимо открыть вкладку SUPPORT, нажать на кнопку REGISTER. Ввести в поле REGISTERED NAME имя: RW9HK, в поле REGISTRATION KEY серийный номер: SSS2-2625702. Далее нажать кнопку UNLOCK. Теперь программа готова к работе. 

» Запустить WinTone

6. SpectInv



Разработка РЦОА ДВРУ. Программа предназначена для инвертирования спектра речевого сигнала с возможностью прослушивания результатов преобразования. Крайне удобная и простая в обращении программа. Инсталляции не требует. 

» Запустить программу

7. Комплекс программ для спектральной инверсии и частотного сдвига

Комплекс программ, предназначенных для инвертирования и сдвига спектра сигнала. Графического интерфейса не имеется, поддерживает MS-DOS, набор команд осуществляется из командной строки. Необходимо использовать вместе с программой CoolEdit. Инсталляции не требует. Программа находится в каталоге: Signal database\Programms\Distributive\Finv 

 

8. TFM



Программный комплекс TFM предназначен для демодуляции и декодирования в реальном времени передач стандартов MIL-STD-188-110A и STANAG-4285. Для работы комплекса не требуется никаких дополнительных технических средств, кроме ПЭВМ со звуковой платой. Инсталляции не требуется. 

» Запустить программу

9. Poc32



Предназначение данной программы: осуществление перехвата пэйджинговых сообщений по протоколам FLEX и POCSAG. Возможна работа как с подключенным к ПЭВМ РПУ, либо с уже записанными файлами. Инсталляции не требует. Локализована на русский язык. 

» Запустить программу

10. PC Ale



Программа предназначена для декодирования многопозиционных передач, в частности стандарта MIL-STD-188-141A (ALE). Работает непосредственно с РПУ или с записанным сигналом. Наглядная выдача результатов. Возможность ведения лог-файлов. Инсталляции не требует. 

» Запуск программы 

11. Аншлаг



Программа предназначена для анализа, определения скорости, демодуляции ЧМ-2 передач, записанных в формате AIF. Данное преобразование возможно осуществить с помощью программ CoolEdit или SpectraPlus. Результат выдается в виде бинарного файла. Установки не требует. 

» Запустить программу

12.MFSKdem



Программа предназначена для демодуляции многопозиционных ЧМ-передач с числом позиций от 2 до 8. Установки не требует. Разработка РЦОА ДВРУ. 

» Запустить программу

13. Analysis



Разработка Военной Академии Связи (г. Санкт-Петербург). Наиболее полная и качественная программа для анализа сигналов. Имеется возможность просмотра временной диаграммы. Построение сигнального созвездия, глазковых диаграмм. Точное определение скорости передачи. Построение спектра сигнала. Применение к сигналу различных преобразований. Инсталляции не требует. Некоторые особенности и ограничения: обрабатываемые файлы необходимо записывать в тот-же каталог, что и сама программа, при перезаписи в этот каталог файла для анализа ему необходимо присвоить расширение .BIN, всего могут быть помещены для анализа 8 файлов и кроме того для полноценной работы необходимо перевести формат файла из INTEGER во FLOAT в меню обработка под другим именем. Еще одно ограничение - программа написана под  DOS. В некоторых версиях Windows могут возникнуть проблемы с отображением русских шрифтов. Тогда необходимо сначала запустить из каталога Signal Database\Programms\Distributive\Analysis программу русских шрифтов RK.COM, а затем уже сам анализатор. 

» Запустить программу     »Прочитать руководство
